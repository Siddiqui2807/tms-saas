# TMS SaaS - Timesheet Management System

## Features
- JWT Authentication
- Role Based Access Control
- Admin / HR / Manager / Employee roles
- Task & Timeline Management
- Approval Workflow
- Activity Logs
- Dashboard Analytics
- Modern React UI

## Tech Stack
Frontend: React + Vite + Tailwind
Backend: Django + DRF
Database: SQLite
Auth: JWT

## Run Project
Backend:
python manage.py migrate
python manage.py runserver

Frontend:
npm install
npm run dev
